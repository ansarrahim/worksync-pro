# QUICK START - WorkSync Pro

WorkSync Pro now uses PostgreSQL so the whole team can use one shared online database.

## Local Development

1. Create backend environment file:

```bash
copy backend\.env.example backend\.env
```

2. Set `DATABASE_URL` in `backend\.env` to a PostgreSQL connection string.

3. Start everything from the project root:

```bash
npm start
```

This starts:

- Backend: http://localhost:5000
- Frontend: http://localhost:3000
